Setting Up:

1. Run the NodeJS servers on same or different machines.
2. Logic Server runs on port no. 9658
3. Filter Server runs on port no. 19620
4. RegRepoSec Server runs on port no. 5000
5. After running all the servers, verify that the URL for different machines is reachable.
6. Install Sensor and Gateway app on Android Devices and then pair them using Bluetooth.
7. The databases are already created on a system with mongodb installed when the servers are run.
8. Verify the system using the UserAppTest android application. It host all the features to test the app.
